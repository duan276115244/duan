## 🚀 三分钟上手

```bash
# 1. 克隆并安装
git clone <repo-url> && cd jws
npm install

# 2. 配置环境变量（至少一个模型 API Key）
cp .env.example .env
# 编辑 .env，推荐 DeepSeek（性价比最高）

# 3. 启动（三选一）
npm run dev          # CLI 终端
npm run dev:full     # Web 控制台 → http://localhost:3001
npm run dev:desktop  # Electron 桌面端
```
