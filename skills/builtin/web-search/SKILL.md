# 网络搜索技能

## 适用场景
- 获取实时信息
- 技术调研
- 资料查找

## 执行步骤
1. 使用 web_search 工具搜索关键词
2. 使用 web_fetch 工具抓取相关网页内容
3. 提取关键信息并整理
4. 引用来源链接

## 输出规范
- 提供信息来源
- 区分事实和观点
- 标注信息时效性
