# 项目初始化技能

## 适用场景
- 新项目脚手架搭建
- 项目配置文件生成
- 目录结构规划

## 执行步骤
1. 确定项目类型和技术栈
2. 创建项目目录结构
3. 生成配置文件（package.json/tsconfig.json 等）
4. 安装依赖
5. 创建示例代码

## 支持的项目类型
- React / Vue / Angular
- Node.js (Express / Fastify / NestJS)
- Python (FastAPI / Flask / Django)
- Go / Rust / Java
