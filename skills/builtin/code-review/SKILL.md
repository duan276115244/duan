# 代码审查技能

## 适用场景
- 代码质量评估
- 安全漏洞检测
- 最佳实践检查

## 执行步骤
1. 使用 file_read 读取目标代码文件
2. 分析代码结构、命名规范、错误处理
3. 检查常见安全漏洞（注入、XSS、硬编码密钥等）
4. 输出审查报告和改进建议

## 审查维度
- 安全性：注入、XSS、敏感信息泄露
- 可读性：命名、注释、代码结构
- 性能：算法效率、内存使用
- 可维护性：模块化、耦合度
