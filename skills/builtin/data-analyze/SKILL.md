# 数据分析技能

## 适用场景
- 数据清洗和处理
- 统计分析
- 数据可视化
- 报表生成

## 执行步骤
1. 使用 file_read 读取数据文件
2. 使用 code_execute 编写分析脚本（Python）
3. 输出分析结果和图表
4. 生成分析报告

## 支持的数据格式
- CSV
- JSON
- Excel (.xlsx)
- 纯文本
