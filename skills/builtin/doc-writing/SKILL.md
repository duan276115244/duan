# 文档撰写技能

## 适用场景
- 项目文档生成
- API 文档编写
- 技术方案撰写
- 用户手册

## 执行步骤
1. 分析文档需求
2. 使用 file_read 收集相关代码/配置信息
3. 使用 file_write 生成文档文件
4. 格式化输出（Markdown）

## 文档类型
- README.md
- API 文档
- 架构设计文档
- 部署指南
- 用户手册
